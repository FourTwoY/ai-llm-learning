# Day 35 - 把检索结果喂给千问生成答案

## 今天完成的内容

1. 输入用户问题
2. 对 query 做 embedding
3. 检索 top-3 相关 chunk
4. 将检索结果拼成上下文
5. 把上下文和问题一起送给千问生成答案
6. 输出回答并附带参考来源

## 使用的模型

- Embedding: text-embedding-v4
- Chat: qwen3-max-2026-01-23

## 本次练习收获

- 完成了 RAG 的第一次闭环
- 理解了“检索 + 生成”是如何串起来的
- 学会了约束模型尽量只根据材料回答
- 知道了如何在回答中附带引用来源
